# Disable wakeup from suspend usb devices

## Content

`disable_wakeup_from_suspend_usb_devices.sh`: script for disabling the usb devices
`disable_wakeup_from_suspend_usb_devices.service`: service for disabling usb devices
`install_service.sh`: Install the service automatically to disable devices on every reboot

## Instructions

Run `install_service.sh` and done
